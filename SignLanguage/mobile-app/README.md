# Expo Go 实验版（手语实时翻译）

这是一个基于 Expo Go 的移动端实验应用，用手机摄像头实时采集手语动作，并通过 WebSocket 调用后端 `/ws` 接口，返回“手语动作 -> 文字”实时翻译结果。

## 1. 安装依赖

```bash
npm install
```

## 2. 启动后端（电脑）

在项目根目录运行（示例）：

```bash
python backend/main.py --config configs/config.yaml --host 0.0.0.0 --port 8000
```

> 注意：`--host 0.0.0.0` 是为了让手机能通过局域网访问电脑。

## 3. 启动 Expo（电脑）

在 `mobile-app` 目录运行：

```bash
npm run start
```

用手机 Expo Go 扫码打开。

## 4. 配置移动端后端地址

先在电脑上查看局域网 IPv4：

```bash
ipconfig
```

在输出里找到 “IPv4 地址”（示例：`10.100.102.85`），然后在 App 顶部输入框填写：

```text
ws://<你电脑局域网IP>:8000/ws
```

例如：

```text
ws://10.100.102.85:8000/ws
```

然后点击“开始识别”。

## 5. 当前能力

- 实时摄像头采样（前置）
- 周期抓帧并发送到后端
- 实时显示翻译文字、置信度、类别 ID
- 展示 Top-3 候选与原始后端消息

## 6. 说明

- 这是实验版方案，当前是“拍照轮询 + WebSocket”模式，优先验证端到端可用性。
- 若要进一步提速，可改成更高效的视频帧流方案（例如原生模块或服务端优化）。

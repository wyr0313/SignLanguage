## 运行指南

### 1) 安装环境

建议 Python 3.10+。

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) 提取关键点特征（首次训练必须执行）

```bash
python scripts\extract_mediapipe_features.py --config configs\config.yaml
```

### 3) 训练 Transformer 模型

```bash
python scripts\train_transformer.py --config configs\config.yaml
```

### 4) 启动后端服务（含前端页面）

本机浏览器调试（仅本机可访问）：

```bash
python backend\main.py --config configs\config.yaml --host 127.0.0.1 --port 8000
```

手机 / 局域网内其它设备连 WebSocket 时，**必须用 `0.0.0.0`**，否则手机访问 `ws://<电脑IP>:8000/ws` 会一直断开：

```bash
python backend\main.py --config configs\config.yaml --host 0.0.0.0 --port 8000
```

若 Windows 弹出防火墙提示，请允许专用网络访问；仍不通可在「高级安全 Windows 防火墙」里为 8000 端口添加入站规则。

浏览器打开：

- `http://127.0.0.1:8000/`（本机）或 `http://<电脑局域网IP>:8000/`（同网其它设备）

### 5) 启动移动端实验版（Expo Go）

```bash
cd mobile-app
npm install
npm run start
```
#### （可选）离线模式启动 Expo（更省事）(要和电脑连同一个网，切记)

```bash
cd D:\SignLanguage\mobile-app
npx expo start --offline
```

手机在 Expo Go App 顶部地址栏填写（先在电脑上查局域网 IPv4）：
在输出里找到 “IPv4 地址”，然后填写：

- `ws://<电脑局域网IP>:8000/ws`（示例：`ws://10.100.102.85:8000/ws`）

然后点击“开始识别”即可进行移动端实时识别。

```bash
ipconfig
```

然后填写：

- `ws://<电脑局域网IP>:8000/ws`
